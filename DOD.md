# Definition of Done (Free-First)

Bu layihə yalnız pulsuz texnologiyalarla buraxılır.

## Məcburi keyfiyyət şərtləri

- [ ] Bütün `must-have` acceptance criteria `PASS`.
- [ ] `index.html` linter xətasızdır.
- [ ] Runtime yoxlaması (`node` ilə script compile) uğurludur.
- [ ] Bloker və kritik bug sayı `0`.
- [ ] Həftə açılma qaydası (70%) düzgün işləyir.
- [ ] Reflection yaşa uyğun həddlə işləyir (8-10: 30, 11-12: 50).
- [ ] Focus Session timer start/stop/reset stabil işləyir.
- [ ] Import/Export JSON validasiya ilə işləyir.
- [ ] Sertifikat yalnız şərt ödəndikdə aktivdir.
- [ ] Mobil ekranlarda (320px+) istifadə oluna bilir.
- [ ] Əsas UI kontrastı və oxunaqlılığı WCAG 2.2 AA səviyyəsinə yaxındır.

## Free stack qaydası

- [ ] Framework yoxdur (vanilla JS/CSS/HTML).
- [ ] Ödənişli API və SDK yoxdur.
- [ ] Ödənişli analytics yoxdur.
- [ ] Hosting tələb etmir (offline işləyir).
- [ ] Bütün istifadə olunan resurslar pulsuz və açıq girişlidir.

## Child-safe qaydası

- [ ] Toksik gamification elementləri yoxdur (leaderboard, streak pressure, XP addiction).
- [ ] AI cavablarının yoxlanması barədə açıq xəbərdarlıq var.
- [ ] Valideyn panelində nəzarət və həftəlik xülasə var.
