# Acceptance Criteria Matrix

## 1) Onboarding

- AC-ONB-01: İstifadəçi ad daxil edəndən sonra dashboard açılır.
- AC-ONB-02: Yaş (8-12) və avatar localStorage-da saxlanır.
- AC-ONB-03: Yenidən açıldıqda profil bərpa olunur.

## 2) Progress & Unlock

- AC-PRG-01: Tapşırıq seçimi progress faizini yeniləyir.
- AC-PRG-02: Reflection limiti dolanda progress artır.
- AC-PRG-03: Həftə yalnız əvvəlki həftə 70% olanda açılır.

## 3) Reflection

- AC-RFL-01: 8-10 yaş üçün min 30 simvol.
- AC-RFL-02: 11-12 yaş üçün min 50 simvol.
- AC-RFL-03: Reflection autosave ilə itmir.

## 4) Focus Session

- AC-FCS-01: Start timer azaldır.
- AC-FCS-02: Stop timeri dayandırır.
- AC-FCS-03: Reset yaşa uyğun default dəqiqəyə qaytarır.

## 5) Project Block (Week 9-10)

- AC-PRJ-01: 4 şablondan seçim mümkündür.
- AC-PRJ-02: Seçilən layihə dashboard/hafta səhifəsində görünür.
- AC-PRJ-03: Layihə qeydləri saxlanır.

## 6) Parent Panel

- AC-PAR-01: Weekly summary doğru göstəricilər verir.
- AC-PAR-02: Checklist checkbox-ları saxlanır.
- AC-PAR-03: Tövsiyə mətnləri görünür.

## 7) Data Safety

- AC-DAT-01: Export `.json` faylı endirilir.
- AC-DAT-02: Import yalnız valid struktur qəbul edir.
- AC-DAT-03: Invalid JSON üçün xəbərdarlıq çıxır.

## 8) Certificate

- AC-CER-01: Şərt ödənmədən düymə disabled olur.
- AC-CER-02: Şərt ödənəndə sertifikat faylı yaradılır.

## 9) Accessibility & UX

- AC-UX-01: 320px ekranda elementlər sıxışmır.
- AC-UX-02: Kontrast/oxunaqlılıq istifadə oluna bilən səviyyədədir.
- AC-UX-03: Naviqasiya düymələri bütün əsas ekranlarda mövcuddur.
